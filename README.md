# 🐾 VetSystem - Sistema de Gestión Veterinaria

## Descripción
VetSystem es una API REST desarrollada con Spring Boot que permite gestionar de forma digital los procesos de una veterinaria, incluyendo el registro de mascotas, dueños, veterinarios, citas médicas, tratamientos y más.
